Política de Privacidade

Este aplicativo utiliza autenticação TikTok para permitir que o próprio usuário envie vídeos para sua conta.

Os dados acessados são usados apenas para autenticação e envio de conteúdo autorizado pelo usuário.

O aplicativo não vende dados pessoais a terceiros.

Metadados locais, como título, legenda e hashtags, podem ser armazenados no computador do usuário para organização da fila de envio.

Se tiver dúvidas, entre em contato pelo e-mail informado pelo responsável do aplicativo.
